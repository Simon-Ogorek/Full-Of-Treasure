# Full Of Treasure (Tentative Name)
An early in development project for a isometric pirate game involving ship battles and Legend of Zelda style combat to get as much treasure in a given run with progressive difficulty

# Goals
- Entity System ✅
- Basic Isometric Tiling System ✅
- 2D Camera ⚠️
- Map Editor ❌
- Player System ❌
- Ship System ❌
- More as I get to them

__Reach Goal: Port this to the Nintendo 3DS__

# Technical
This project is based upon the gameframework2d game engine core, which itself uses SDL2.0 for the purposes as a class project for 2D Game Programming at NJIT

*   **Languages Involved**
    *   C
    *   JSON
        *   For data driven design
    *   Make ( Porting this to CMake )
